问题解决：JMeter安装和使用

一、安装和使用

  1. 首先要确保有 java 环境

  2. 进入官网Apache JMeter - Apache JMeter™ 点击左侧 download releases，点击Binaries下的下载包，win环境选择 zip 结尾的

  3. 解压到任意目录

  4. 配置环境变量，添加系统变量

    • JMETER_HOME：值为压缩包路径
    • CLASSPATH：没有就创建，值为%JMETER_HOME%\lib\ext\ApacheJMeter_core.jar; %JMETER_HOME%\lib\jorphan.jar;，有就在后面追加值，注意加个分号

  5. 打开方式：

    • 直接打开解压目录/bin下的ApacheJMeter.jar，不过如果安装的 java 方式不同，有的可能直接打开不了
    • 直接打开解压目录/bin下的jmeter.bat，命令行和GUI界面会保持同时打开和关闭

二、汉化

找到解压目录下的bin目录，打开jmeter.properties 文件

找到language行，将值改为language=zh_CN

去掉前面的#，以后打开就是中文界面了
